To connect to the Staging Redis instance on AWS:
	Staging: `./redis-tunnels.sh staging`
	Production: `./redis-tunnels.sh production`

Go into RedisInsight and connect to the Redis instance:
	Staging: `localhost:6380`
	Production: `localhost:6381`

Password: find in AWS
Username: `default`
Turn on TLS
Make sure local Redis is stopped

To start local redis, run `brew services start redis`
